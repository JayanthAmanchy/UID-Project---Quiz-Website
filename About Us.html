About Our Quiz Website

Our quiz website is built using HTML, CSS, and JavaScript to provide users with an interactive learning experience through multiple-choice questions, instant feedback, and score tracking.

Project Overview

This platform allows users to answer quiz questions from different categories such as Science, Technology, Sports, and Math. It is designed to make learning fun, engaging, and easy to use.

Key Features

  • Multiple-choice quiz questions
  • Timer-based quizzes
  • Instant answer feedback
  • Automatic score calculation
  • Responsive design for mobile and desktop

User Roles

  • Guest User: Can attempt quizzes with limited features
  • Registered User: Can save scores, track progress, and view history
  • Admin: Can manage questions, users, categories, and difficulty levels
