# UID Project--Quiz-Website  
This is a modern interactive quiz platform developed using HTML, CSS, and JavaScript.  
It provides:  
1. Subject-based quizzes  
2. Real-time score tracking  
3. Responsive design  
4. Animated UI  
5. Login/signup  
6. Authentication
